Démarrage de Spoggy

Chargement des éléments clients, Vérifiez que javascript est activé dans votre navigateur. (Could not render the custom element. Check that JavaScript is enabled).

Spoggy utilise Polymer 2 & les Web Components, mais ceux-ci ne sont pas pris en compte par tous les navigateurs Web

Spoggy fonctionne avec les dernières versions de Chrome et Opéra.
Le chargement est lent avec Firefox.
Vous pouvez vérifier la compatibilité de votre navigateur au bas de cette page


Une version dégradée de Spoggy (sans Polymer) existe aussi mais n'est peut-être pas à jour : SPOGGY-STANDALONE


Vous pouvez obtenir plus d'informations sur Spoggy ou contribuer à son développement sur https://github.com/scenaristeur/heroku-spoggy
